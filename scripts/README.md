# Scripts de Utilidad

## Migraciones de Supabase

### `apply-migration-interactive.ps1`
Script interactivo con menú para aplicar migraciones:
- Opción 1: Aplicar directamente con Supabase CLI
- Opción 2: Mostrar SQL para copiar manualmente
- Opción 3: Abrir Supabase Dashboard

```powershell
.\scripts\apply-migration-interactive.ps1
```

### `apply-migration.ps1`
Script directo para aplicar migración con Supabase CLI:

```powershell
.\scripts\apply-migration.ps1
```
